# Ledger

## Purpose/Goals

Ledger is a simple budgeting application that:
- Allows a user to connect with their bank
- Allows a user to create purchase categories and categorize purchases
- Allows a user to see a monthly summary of their purchases
- Notifies a user when a purchase is made and asks them to categorize the purchase
- Allows a user to join/create a budgeting group, where multiple users' purchases are combined and can be categorized

The purpose of this application is to provide a quick and easy way for users to stay on top/be aware of their finances month-to-month.
